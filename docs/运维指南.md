# SRW DeepSeek 本地桌面网关运维指南

## 1. 角色与职责

### 管理员

- 维护管理员密码
- 配置上游 DeepSeek API Key
- 设定和调整月度预算
- 导出月报或授权他人导出
- 处理超额和异常情况

### 终端用户

- 正常使用 VS Code DeepSeek Copilot Chat
- 发现异常时及时反馈管理员

## 2. 安全注意事项

- 上游 DeepSeek API Key 通过 Windows 系统级加密保存在本地，请勿通过聊天工具等明文传播。
- 管理员密码建议使用强密码，避免与个人账户密码复用。
- 每次预算调整建议记录变更原因和时间。

## 3. 月度操作流程

### 月初

管理员确认每台设备的月度预算是否需要调整，记录调整原因和生效日期。

### 月中

建议每周查看一次当月累计费用。如发现异常高费用，及时核查模型和使用情况。

### 月末

1. 在 GUI 的"报表"页选择对应月份，导出 CSV 或 XLSX 明细报表。
2. 汇总各设备报表，标记异常记录。
3. 提交月度费用汇总。

## 4. 常见异常处理

### 上游接口异常

**现象：** 请求失败，返回 401、429 或 5xx。

**处理步骤：**
1. 检查上游 API Key 是否失效（在"设置"页重新录入）。
2. 检查 DeepSeek 服务状态。
3. 检查本机网络连通性。

### 本地数据库异常

**现象：** 无法写入请求日志或报表导出失败。

**处理步骤：**
1. 在"运维"页备份当前数据。
2. 检查磁盘可写权限。
3. 如问题持续，可在"运维"页重置数据库（需要管理员密码）。

### 配置校验失败

**现象：** 程序启动时报配置完整性错误。

**处理步骤：**
1. 判断是否有人直接修改了配置文件。
2. 从备份恢复数据（"运维"页 → 恢复数据）。
3. 如无备份，由管理员在"设置"页重新初始化配置。

## 5. 数据备份建议

- 建议定期在"运维"页使用备份功能生成 ZIP 文件。
- 也可以直接备份整个程序目录（含 EXE、配置、数据库、日志等）。
- 保留每月的导出报表作为审计材料。
